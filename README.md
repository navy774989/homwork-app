# Getting Started

To get a local copy up and running follow these simple steps.

## Installation

1. Clone the repo \
   `git@github.com:navy774989/homwork-app.git`

2. Install NPM packages \
   `yarn install`

3. Build the app \
   `yarn run build`
4. Start the Server
   `yarn global add serve && 
serve -s build`
